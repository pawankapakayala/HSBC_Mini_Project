HSBC (Hostel Student Banking Card) is a **C++ console-based mini project** designed to simulate a **cashless transaction system for hostel students**.  
The system allows students to use a virtual card for purchases at hostel facilities such as bakeries, stationery stores, and essential item counters.

This project was developed as part of academic coursework to demonstrate **Object-Oriented Programming (OOP) concepts in C++** and real-world problem modeling.
